# 🍽️ Recipe Finder App

A modern Android recipe finder application built with **Kotlin**
and **MVVM Architecture** using the free TheMealDB API.

## ✨ Features

- 🔍 **Search Recipes** - Real-time search with debounce
- 📂 **Category Filtering** - Browse recipes by food categories
- 📖 **Detailed View** - Full recipe with ingredients & instructions
- ▶️ **YouTube Integration** - Watch recipe videos directly
- 🔄 **Pull to Refresh** - Swipe down to reload content
- ⚡ **Fast Image Loading** - Async image loading with Glide

## 🏗️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Kotlin | Primary Language |
| MVVM + Repository | Architecture Pattern |
| Retrofit | REST API calls |
| Coroutines | Async/Non-blocking operations |
| LiveData | Reactive UI updates |
| ViewModel | UI state management |
| Glide | Async image loading |

| RecyclerView | Efficient list rendering |
